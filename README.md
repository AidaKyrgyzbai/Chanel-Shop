# CHANEL --- Rare Collection

CHANEL --- Rare Collection --- это одностраничный (SPA)
интернет-магазин, вдохновлённый эстетикой бренда Chanel, предназначенный
для демонстрации и продажи редких, архивных и лимитированных сумок. Сайт
предоставляет пользователю возможность просматривать каталог товаров с
динамической генерацией карточек, использовать поиск, фильтрацию и
сортировку по различным параметрам.

При выборе товара открывается модальное окно с галереей изображений,
описанием и деталями. Реализованы функции добавления в корзину и
избранное, а также базовая система аккаунта с регистрацией и входом.
Оформление заказа включает ввод данных, выбор способа оплаты и
подтверждение.

Проект разработан без использования фреймворков на HTML5, CSS3 и чистом
JavaScript, с применением Flexbox и Grid для адаптивной верстки,
CSS-анимаций и динамической работы через DOM. Данные о товарах хранятся
в виде массива внутри JavaScript, что делает сайт полностью статичным и
простым для запуска --- достаточно открыть файл index.html в браузере.
